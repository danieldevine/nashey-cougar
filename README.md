#Nashey Cougar


Chrome extension that replaces occurrences of 'Nicolas Cage' with 'Nashey Cougar'

This is "inspired" by [cloud-to-butt](https://github.com/panicsteve/cloud-to-butt) by [Steven Frank](https://github.com/panicsteve)

##all because
[of this](https://www.youtube.com/watch?v=kX30tdH0xhs)

##Installation
In Chrome, choose Window > Extensions.
Drag nashey-cougar.crx into the page that appears.
